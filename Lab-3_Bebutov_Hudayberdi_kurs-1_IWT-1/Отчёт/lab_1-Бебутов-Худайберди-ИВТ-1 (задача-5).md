## Задача 1.5 – Двусвязный список 

### Постановка задачи
Реализовать в виде структур двунаправленный связный список и совершить отдельно его обход в прямом и обратном направлениях с распечаткой значений каждого элемента списка.

### Математическая модель
Двусвязный список — структура данных, состоящая из узлов, каждый из которых содержит:
- данные (целое число)
- указатель на следующий узел
- указатель на предыдущий узел

Обход в прямом направлении: от головы к хвосту через указатель next.
Обход в обратном направлении: от хвоста к голове через указатель prev.

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| Node           | struct     | узел списка |
| data           | int        | данные узла |
| next           | struct Node* | указатель на следующий узел |
| prev           | struct Node* | указатель на предыдущий узел |
| head           | struct Node* | указатель на первый узел |
| tail           | struct Node* | указатель на последний узел |
| current        | struct Node* | указатель для обхода |
| n              | int        | количество элементов |
| value          | int        | вводимое значение |
| i              | int        | счётчик цикла |

### Код программы 

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node* next;
    struct Node* prev;
} Node;

int main() {
    Node* head = NULL;
    Node* tail = NULL;
    Node* current = NULL;
    int n, value;
    
    scanf("%d", &n);
    
    for (int i = 0; i < n; i++) {
        scanf("%d", &value);
        Node* newNode = (Node*)malloc(sizeof(Node));
        newNode->data = value;
        newNode->next = NULL;
        newNode->prev = tail;
        
        if (tail != NULL) {
            tail->next = newNode;
        }
        tail = newNode;
        
        if (head == NULL) {
            head = newNode;
        }
    }
    
    current = head;
    while (current != NULL) {
        printf("%d ", current->data);
        current = current->next;
    }
    printf("\n");
    
    current = tail;
    while (current != NULL) {
        printf("%d ", current->data);
        current = current->prev;
    }
    printf("\n");
    
    current = head;
    while (current != NULL) {
        Node* temp = current;
        current = current->next;
        free(temp);
    }
    
    return 0;
}
