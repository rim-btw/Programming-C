## Задача 2.3 – Перечисление для дней недели 

### Постановка задачи
Создайте перечислимый тип данных (enum) для семи дней недели и распечатайте на экране его значения, как целые числа.

### Математическая модель
Перечисление enum автоматически присваивает целые значения, начиная с 0:  
Понедельник = 0, Вторник = 1, Среда = 2, Четверг = 3, Пятница = 4, Суббота = 5, Воскресенье = 6.

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| Weekday        | enum       | перечисление дней недели |
| day            | enum Weekday| переменная для перебора дней |

### Код программы 

```c
#include <stdio.h>

enum Weekday { Понедельник, Вторник, Среда, Четверг, Пятница, Суббота, Воскресенье };

int main() {
    for (enum Weekday day = Понедельник; day <= Воскресенье; day++) {
        printf("%d ", day);
    }
    printf("\n");
    return 0;
}
