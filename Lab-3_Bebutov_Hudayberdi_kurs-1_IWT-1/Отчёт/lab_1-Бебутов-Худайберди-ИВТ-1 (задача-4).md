## Задача 1.4 – Битовые поля для даты 

### Постановка задачи
Используя так называемые "битовые" поля в структуре С, создать экономную структуру в оперативной памяти для заполнения даты некоторого события, например даты рождения человека.

### Математическая модель
- День: от 1 до 31 → требуется 5 бит (2^5 = 32)
- Месяц: от 1 до 12 → требуется 4 бита (2^4 = 16)
- Год: от 0 до 2047 → требуется 11 бит (2^11 = 2048)
- Итого: 5 + 4 + 11 = 20 бит (вместо 96 бит для трёх int)

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| day            | unsigned int:5 | день рождения |
| month          | unsigned int:4 | месяц рождения |
| year           | unsigned int:11 | год рождения |
| birthday       | struct Date | переменная для хранения даты |

### Код программы  

```c
#include <stdio.h>

struct Date {
    unsigned int day   : 5;
    unsigned int month : 4;
    unsigned int year  : 11;
};

int main() {
    struct Date birthday;

    printf("Введите день (1-31): ");
    scanf("%u", &birthday.day);

    printf("Введите месяц (1-12): ");
    scanf("%u", &birthday.month);

    printf("Введите год (0-2047): ");
    scanf("%u", &birthday.year);

    printf("\nДата рождения: %u.%u.%u\n", 
           birthday.day, birthday.month, birthday.year);
    
    printf("Размер структуры: %zu байт\n", sizeof(birthday));

    return 0;
}
