## Задача 1.1 – Факториал: цикл и рекурсия  

### Постановка задачи
Цель: реализовать и сравнить два способа вычисления факториала: итеративный и рекурсивный.  
Вход: целое число n ≥ 0.  
Выход: результаты вычисления факториала двумя способами.  
Требования:  
1. Есть две отдельные функции (через цикл и через рекурсию).  
2. Обе функции для одного и того же n дают одинаковый ответ.  
3. Корректно обработан случай n = 0 (факториал равен 1).  
Пример: вход 5 → обе функции печатают 120.

### Математическая модель
Факториал числа n определяется как:
- n! = 1 × 2 × 3 × ... × n, для n ≥ 1
- 0! = 1

Итеративный метод: произведение чисел от 1 до n в цикле.  
Рекурсивный метод: n! = n × (n-1)!, с базовым случаем 0! = 1.

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| n              | int        | число, для которого вычисляется факториал |
| result         | int        | результат вычисления факториала |
| i              | int        | счётчик цикла |
| factorial_iter | int (функция) | итеративная функция факториала |
| factorial_rec  | int (функция) | рекурсивная функция факториала |

### Код программы  

```c
#include <stdio.h>

int factorial_iter(int n) {
    int result = 1;
    for (int i = 1; i <= n; i++) {
        result = result * i;
    }
    return result;
}

int factorial_rec(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial_rec(n - 1);
}

int main() {
    int n;
    scanf("%d", &n);
    int iter_result = factorial_iter(n);
    int rec_result = factorial_rec(n);
    printf("%d\n", iter_result);
    printf("%d\n", rec_result);
    return 0;
}
