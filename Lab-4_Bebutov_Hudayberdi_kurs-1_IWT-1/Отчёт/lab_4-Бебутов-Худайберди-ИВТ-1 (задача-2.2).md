## Задача 2.2 – Конвертация строк в числа 

### Постановка задачи
Цель: безопасно преобразовывать текст в int и double, чтобы программа корректно реагировала на ошибочный ввод.

Что подать на вход: две строки: одна с целым числом, другая с числом с плавающей точкой.

Что выполнить:
1. для целого числа используйте strtol;
2. для вещественного числа используйте strtod;
3. перед преобразованием обнулите errno;
4. после преобразования проверьте:
   - был ли распознан хотя бы один числовой символ;
   - не произошло ли переполнение диапазона (errno == ERANGE);
   - не остались ли лишние символы после числа.

Что вывести: результат преобразования и понятное сообщение об успешности или причине ошибки.

Мини-пример: " -42" → int -42, "3.14" → double 3.14

### Математическая модель
Преобразование строки в число:
- Для целых: значение = strtol(строка, &endptr, 10)
- Для вещественных: значение = strtod(строка, &endptr)

Условия успешного преобразования:
1. endptr != строка (хотя бы один символ распознан)
2. errno != ERANGE (нет переполнения)
3. *endptr == '\0' или *endptr == '\n' (нет лишних символов)

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| input_line     | char[200]  | строка для ввода двух чисел |
| int_str        | char[100]  | первое число (целое) |
| double_str     | char[100]  | второе число (вещественное) |
| int_end        | char*      | указатель на первый нераспознанный символ для целого |
| double_end     | char*      | указатель на первый нераспознанный символ для вещественного |
| int_val        | long       | преобразованное целое значение |
| double_val     | double     | преобразованное вещественное значение |
| space_pos      | int        | позиция пробела между числами |

### Код программы

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <errno.h>

int main() {
    char input_line[200];
    char int_str[100];
    char double_str[100];
    char *int_end;
    char *double_end;
    long int_val;
    double double_val;
    int space_pos;
    
    printf("Введите целое и вещественное число через пробел: ");
    fgets(input_line, sizeof(input_line), stdin);
    
    input_line[strcspn(input_line, "\n")] = '\0';
    
    for (space_pos = 0; space_pos < (int)strlen(input_line); space_pos++) {
        if (input_line[space_pos] == ' ') {
            break;
        }
    }
    
    if (space_pos == (int)strlen(input_line)) {
        printf("Ошибка: не найден пробел между числами\n");
        return 1;
    }
    
    strncpy(int_str, input_line, space_pos);
    int_str[space_pos] = '\0';
    
    strcpy(double_str, input_line + space_pos + 1);
    
    errno = 0;
    int_val = strtol(int_str, &int_end, 10);
    
    if (int_end == int_str) {
        printf("Ошибка целого числа: цифры не найдены\n");
    } else if (errno == ERANGE) {
        printf("Ошибка целого числа: выход за диапазон\n");
    } else if (*int_end != '\0') {
        printf("Ошибка целого числа: лишние символы после числа\n");
    } else {
        printf("Целое число: %ld\n", int_val);
    }
    
    errno = 0;
    double_val = strtod(double_str, &double_end);
    
    if (double_end == double_str) {
        printf("Ошибка вещественного числа: цифры не найдены\n");
    } else if (errno == ERANGE) {
        printf("Ошибка вещественного числа: выход за диапазон\n");
    } else if (*double_end != '\0') {
        printf("Ошибка вещественного числа: лишние символы после числа\n");
    } else {
        printf("Вещественное число: %f\n", double_val);
    }
    
    return 0;
}
