## Задача 1.4 – Смешанное произведение трёх векторов в 3D  

### Постановка задачи
Цель: вычислять смешанное произведение через разбиение задачи на небольшие понятные функции.  

Вход: три вектора длины 3 в декартовых координатах (x, y, z).  

Обязательные интерфейсы функций:
- `void cross3(const double a[3], const double b[3], double out[3]);` — векторное произведение
- `double dot3(const double a[3], const double b[3]);` — скалярное произведение
- `double triple3(const double a[3], const double b[3], const double c[3]);` — смешанное произведение

Требования:
1. Функция смешанного произведения использует две предыдущие функции, а не дублирует формулы вручную.
2. Смешанное произведение вычисляется по формуле: [a, b, c] = a · (b × c)

Справочная формула для векторного произведения:
b × c = (b_y·c_z - b_z·c_y, b_z·c_x - b_x·c_z, b_x·c_y - b_y·c_x)

Пример: a = (1,0,0), b = (0,1,0), c = (0,0,1) → смешанное произведение = 1

### Математическая модель
Для векторов в трёхмерном пространстве:
- Скалярное произведение: a·b = a_x·b_x + a_y·b_y + a_z·b_z
- Векторное произведение: (b×c) = (b_y·c_z - b_z·c_y, b_z·c_x - b_x·c_z, b_x·c_y - b_y·c_x)
- Смешанное произведение: a·(b×c) — число, равное ориентированному объёму параллелепипеда

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| a              | double[3]  | первый вектор |
| b              | double[3]  | второй вектор |
| c              | double[3]  | третий вектор |
| out            | double[3]  | результат векторного произведения |
| cross_result   | double[3]  | временный вектор для хранения b×c |
| i              | int        | счётчик цикла |
| result         | double     | результат смешанного произведения |

### Код программы  

```c
#include <stdio.h>

void cross3(const double a[3], const double b[3], double out[3])
{
    out[0] = a[1] * b[2] - a[2] * b[1];
    out[1] = a[2] * b[0] - a[0] * b[2];
    out[2] = a[0] * b[1] - a[1] * b[0];
}

double dot3(const double a[3], const double b[3])
{
    return a[0] * b[0] + a[1] * b[1] + a[2] * b[2];
}

double triple3(const double a[3], const double b[3], const double c[3])
{
    double cross_result[3];
    cross3(b, c, cross_result);
    return dot3(a, cross_result);
}

int main()
{
    double a[3], b[3], c[3];
    
    scanf("%lf %lf %lf", &a[0], &a[1], &a[2]);
    scanf("%lf %lf %lf", &b[0], &b[1], &b[2]);
    scanf("%lf %lf %lf", &c[0], &c[1], &c[2]);
    
    double result = triple3(a, b, c);
    
    printf("%.0lf\n", result);
    
    return 0;
}
