## Задача 6 – Квадраты элементов массива 

### Постановка задачи
Цель: работа с одномерным статическим массивом.  
Вход: число n, затем n целых чисел.  
Выход: n чисел — квадраты элементов в том же порядке.  
Ограничения: 1 <= n <= 100, |a[i]| <= 1000.  
Пример:  
Ввод: 4 и 1 -2 3 4  
Вывод: 1 4 9 16

### Математическая модель
Для каждого элемента массива a[i] вычислить b[i] = a[i] × a[i].

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| n              | int        | количество элементов массива |
| a              | int[100]   | статический массив исходных чисел |
| i              | int        | счётчик цикла |

### Код программы 

```c
#include <stdio.h>

int main() {
    int n;
    int a[100];
    
    scanf("%d", &n);
    
    for (int i = 0; i < n; i++)
        scanf("%d", &a[i]);
    
    for (int i = 0; i < n; i++)
        printf("%d ", a[i] * a[i]);
    
    printf("\n");
    
    return 0;
}
