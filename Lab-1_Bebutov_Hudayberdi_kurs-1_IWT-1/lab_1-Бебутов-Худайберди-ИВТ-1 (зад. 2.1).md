## Задача 4 – Максимум из трёх 

### Постановка задачи
Цель: сравнение нескольких значений через if.  
Вход: три целых числа a, b, c.  
Выход: максимальное число.  
Ограничения: |a|, |b|, |c| <= 1000000.  
Пример:  
Ввод: 3 10 7  
Вывод: 10

### Математическая модель
Максимум из трёх чисел:  
max = a, если a >= b и a >= c  
иначе max = b, если b >= a и b >= c  
иначе max = c

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| a, b, c        | int        | три целых числа |
| max            | int        | максимальное из них |

### Код программы  

```c
#include <stdio.h>

int main() {
    int a, b, c;
    int max;
    
    scanf("%d %d %d", &a, &b, &c);
    
    max = a;
    if (b > max) {
        max = b;
    }
    if (c > max) {
        max = c;
    }
    
    printf("%d\n", max);
    
    return 0;
}
