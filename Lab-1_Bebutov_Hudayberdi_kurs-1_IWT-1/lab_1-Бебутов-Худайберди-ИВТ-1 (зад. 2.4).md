## Задача 7 – Разворот массива 

### Постановка задачи
Цель: индексация массива и обмен элементов.  
Вход: число n, затем n целых чисел.  
Выход: элементы в обратном порядке.  
Ограничения: 1 <= n <= 100, |a[i]| <= 1000.  
Пример:  
Ввод: 5 и 10 20 30 40 50  
Вывод: 50 40 30 20 10

### Математическая модель
Дан массив A[0..n-1].  
Результат: A[n-1], A[n-2], ..., A[0]  
Или через обмен: A[i] ↔ A[n-1-i] для i = 0..(n-1)/2

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| n              | int        | количество элементов |
| arr            | int[100]   | статический массив |
| i              | int        | счётчик цикла |
| temp           | int        | временная переменная для обмена |

### Код программы 

```c
#include <stdio.h>

int main() {
    int n;
    int arr[100];
    
    scanf("%d", &n);
    
    for (int i = 0; i < n; i++)
        scanf("%d", &arr[i]);
    
    for (int i = 0; i < n / 2; i++) {
        int temp = arr[i];
        arr[i] = arr[n - 1 - i];
        arr[n - 1 - i] = temp;
    }
    
    for (int i = 0; i < n; i++)
        printf("%d ", arr[i]);
    
    printf("\n");
    return 0;
}
