## Задача 1.2 - Сумма двух чисел

### Постановка задачи
Написать простую программу. Ввести два числа с клавиатуры, вычислить их сумму и напечатать результат. Использовать функцию printf для приглашений на ввод и для распечатки результата. Использовать функцию scanf для ввода каждого числа отдельно с клавиатуры.

### Математическая модель
Сумма двух чисел вычисляется по формуле:

**S = a + b**

где:
- a — первое число
- b — второе число
- S — сумма чисел

### Список идентификаторов

| Имя переменной | Тип данных | Описание |
|----------------|------------|----------|
| number1 | int | Первое число (целое) |
| number2 | int | Второе число (целое) |
| sum | int | Сумма первого и второго числа |

### Код программы

```c
#include <stdio.h>

int main() {
    int number1, number2, sum;
    
    // Запрос первого числа
    printf("Enter first number: ");
    scanf("%d", &number1);
    
    // Запрос второго числа
    printf("Enter second number: ");
    scanf("%d", &number2);
    
    // Вычисление суммы
    sum = number1 + number2;
    
    // Вывод результата
    printf("Sum = %d\n", sum);
    
    return 0;
}
