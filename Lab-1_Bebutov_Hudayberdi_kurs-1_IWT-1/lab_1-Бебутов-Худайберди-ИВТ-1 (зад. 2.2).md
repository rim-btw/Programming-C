# Комплект 2: Организация циклов. Условные конструкции

## Задача 2.1 - Координаты планеты Марс относительно Земли

### Постановка задачи
Вычислить используя цикл for координаты планеты Марс относительно Земли с течением времени t. Распечатать на экране координаты для каждой итерации по t.

Координаты задаются формулами:

**x = r₁·cos(w₁·t) - r₂·cos(w₂·t)**

**y = r₁·sin(w₁·t) - r₂·sin(w₂·t)**

**w₁ = 2π/T₁**

**w₂ = 2π/T₂**

где:
- r₁ — радиус орбиты Марса
- r₂ — радиус орбиты Земли
- T₁ — период обращения Марса
- T₂ — период обращения Земли
- t — время

### Математическая модель

| Параметр | Значение | Единица измерения |
|----------|----------|-------------------|
| r₁ (Марс) | 227.9 | млн км |
| r₂ (Земля) | 149.6 | млн км |
| T₁ (Марс) | 687 | дни |
| T₂ (Земля) | 365 | дни |

**Угловые скорости:**
- w₁ = 2 · 3.14159 / 687 ≈ 0.00915 рад/день
- w₂ = 2 · 3.14159 / 365 ≈ 0.01721 рад/день

**Координаты в момент времени t:**
- x(t) = r₁·cos(w₁·t) - r₂·cos(w₂·t)
- y(t) = r₁·sin(w₁·t) - r₂·sin(w₂·t)

### Список идентификаторов

| Имя переменной | Тип данных | Описание |
|----------------|------------|----------|
| r1, r2 | double | Радиусы орбит Марса и Земли |
| T1, T2 | double | Периоды обращения Марса и Земли |
| w1, w2 | double | Угловые скорости |
| t | double | Время (дни) |
| x, y | double | Координаты Марса относительно Земли |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    // Радиусы орбит (млн км)
    double r1 = 227.9;   // Марс
    double r2 = 149.6;   // Земля
    
    // Периоды обращения (дни)
    double T1 = 687.0;   // Марс
    double T2 = 365.0;   // Земля
    
    // Угловые скорости
    double w1 = 2 * M_PI / T1;
    double w2 = 2 * M_PI / T2;
    
    double t, x, y;
    
    // Заголовок таблицы
    printf("==============================================\n");
    printf("  t (дни)     x (млн км)     y (млн км)\n");
    printf("==============================================\n");
    
    // Цикл по времени от 0 до 800 дней с шагом 100
    for (t = 0; t <= 800; t += 100) {
        x = r1 * cos(w1 * t) - r2 * cos(w2 * t);
        y = r1 * sin(w1 * t) - r2 * sin(w2 * t);
        printf("  %.0f          %.2f          %.2f\n", t, x, y);
    }
    
    printf("==============================================\n");
    
    return 0;
}
