## Задача 3 – Проверка чётности 

### Постановка задачи
Цель: использовать условный оператор if.  
Вход: целое число n.  
Выход: EVEN, если число чётное, иначе ODD.  
Ограничения: |n| <= 1 000 000.  
Пример:  
Ввод: 9  
Вывод: ODD

### Математическая модель
Число n чётное, если остаток от деления на 2 равен 0:  
n % 2 == 0 → EVEN  
иначе → ODD

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| n              | int        | целое число для проверки |

### Код программы 

```c
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    
    if (n % 2 == 0) {
        printf("EVEN\n");
    } else {
        printf("ODD\n");
    }
    
    return 0;
}
