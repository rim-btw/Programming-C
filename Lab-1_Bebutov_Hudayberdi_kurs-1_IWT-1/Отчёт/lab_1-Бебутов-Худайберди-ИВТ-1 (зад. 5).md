## Задача 5 – Сумма от 1 до n 

### Постановка задачи
Цель: цикл for.  
Вход: целое число n.  
Выход: сумма 1 + 2 + … + n.  
Ограничения: 1 <= n <= 10000.  
Пример:  
Ввод: 5  
Вывод: 15

### Математическая модель
Сумма арифметической прогрессии от 1 до n:  
S = 1 + 2 + … + n = n × (n + 1) / 2

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| n              | int        | верхняя граница суммы |
| sum            | int        | вычисленная сумма |

### Код программы  

```c
#include <stdio.h>

int main() {
    int n, sum;
    
    scanf("%d", &n);
    
    sum = n * (n + 1) / 2;
    
    printf("%d\n", sum);
    
    return 0;
}
