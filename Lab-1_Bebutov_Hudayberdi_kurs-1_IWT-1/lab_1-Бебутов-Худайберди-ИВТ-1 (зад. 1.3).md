## Задача 2 – Простая формула 

### Постановка задачи
Цель: вычислить математическое выражение с double.  
Формула: u = (x + y) / 2 + x * y  
Вход: два вещественных числа x, y.  
Выход: значение u с точностью 3 знака после запятой.  
Ограничения: -100.0 <= x, y <= 100.0.  
Пример:  
Ввод: 2.0 4.0  
Вывод: 11.000

### Математическая модель
u = (x + y) / 2 + x * y  
где x, y — вещественные числа.

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| x              | double     | первое число |
| y              | double     | второе число |
| u              | double     | результат вычисления |

### Код программы 

```c
#include <stdio.h>

int main() {
    double x, y, u;
    scanf("%lf %lf", &x, &y);
    u = (x + y) / 2.0 + x * y;
    printf("%.3lf\n", u);
    return 0;
}
