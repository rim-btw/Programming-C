## Задача 1 – Сумма двух чисел 

### Постановка задачи
Цель: отработать ввод/вывод и переменные типа int.  
Вход: два целых числа a и b.  
Выход: их сумма.  
Ограничения: -10000 <= a, b <= 10000.  
Пример: Ввод: 7 5 → Вывод: 12

### Математическая модель
S = a + b

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| a, b           | int        | исходные числа |
| sum            | int        | сумма a и b |

### Код программы 

```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    int sum = a + b;
    printf("%d\n", sum);
    return 0;
}
