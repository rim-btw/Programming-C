## Задача 6 – Побайтовый вывод int 

### Постановка задачи
Цель: понять, как данные хранятся в памяти.  
Дано: int a = 1234567890;  
Выход: значения байтов переменной (в десятичном виде).  
Требование: использовать указатель unsigned char *.

### Математическая модель
Целое число в памяти занимает sizeof(int) байт.  
Каждый байт представляется как целое число от 0 до 255.  
Порядок байт зависит от архитектуры (little-endian / big-endian).

### Список идентификаторов

| Имя переменной | Тип данных        | Смысловое обозначение           |
|----------------|-------------------|----------------------------------|
| a              | int               | исходное число (1234567890)      |
| ptr            | unsigned char*    | указатель на байты переменной a  |
| i              | int               | счётчик цикла                    |

### Код программы 

```c
#include <stdio.h>

int main() {
    int a = 1234567890;
    unsigned char *ptr = (unsigned char*)&a;
    for (int i = 0; i < sizeof(a); i++)
        printf("%d ", ptr[i]);
    printf("\n");
    return 0;
}
