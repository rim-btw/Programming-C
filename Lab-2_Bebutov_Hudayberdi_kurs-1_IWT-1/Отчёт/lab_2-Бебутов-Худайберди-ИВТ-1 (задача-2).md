## Задача 2 – Обмен двух чисел 

### Постановка задачи
Цель: изменить данные по адресу (swap).  
Вход: два целых числа a, b.  
Выход: сначала исходные, затем обменённые значения.  
Требование: обмен выполнять через указатели.  
Пример: Ввод: 3 7  
Вывод: 3 7 и затем 7 3

### Математическая модель
Обмен значениями двух переменных через временную переменную:  
temp = a, a = b, b = temp

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| a              | int        | первое число           |
| b              | int        | второе число           |
| pa             | int*       | указатель на a         |
| pb             | int*       | указатель на b         |
| temp           | int        | временная переменная   |

### Код программы 

```c
#include <stdio.h>

int main() {
    int a, b;
    int *pa = &a, *pb = &b;
    scanf("%d %d", pa, pb);
    printf("%d %d\n", *pa, *pb);
    int temp = *pa;
    *pa = *pb;
    *pb = temp;
    printf("%d %d\n", *pa, *pb);
    return 0;
}
