## Задача 1 – Сложение через указатели 

### Постановка задачи
Цель: получить значение переменных через указатели.  
Вход: два целых числа a, b.  
Выход: сумма чисел.  
Требование: складывать значения через указатели int *pa, int *pb.  
Пример: Ввод: 8 9 → Вывод: 17

### Математическая модель
Сумма двух чисел:  
S = a + b  
где a и b — целые числа.

### Список идентификаторов

| Имя переменной | Тип данных | Смысловое обозначение |
|----------------|------------|------------------------|
| a              | int        | первое число           |
| b              | int        | второе число           |
| pa             | int*       | указатель на a         |
| pb             | int*       | указатель на b         |

### Код программы 

```c
#include <stdio.h>

int main() {
    int a, b;
    int *pa = &a, *pb = &b;
    scanf("%d %d", pa, pb);
    printf("%d\n", *pa + *pb);
    return 0;
}
